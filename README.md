# 🌦 WeatherHub

A modern weather dashboard built with React.

## Features

- Real-Time Weather
- 7-Day Forecast
- Current Location
- Glassmorphism UI
- Responsive Design
- Dark Theme

## Tech Stack

- React
- JavaScript
- Axios
- Bootstrap
- CSS3

## Installation

```bash
npm install
npm start
```
